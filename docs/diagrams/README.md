# Diagrams

This directory contains a number of useful reference diagrams for using pikchr.

- `autochop09.pikchr`: crisp lines and directional flow from concentric center rings.
- `expr.pikchr`: "railroad diagram" similar to those on json.org and sqlite.org
- `real01.pikchr`: colorized view of data payload
- `test38b.pikchr`: compiler flow
- `test55.pikchr`: component separation and flow
- `test58.pikchr`: rebase vs. merge diagram
- `test71.pikchr`: detailed flow chart
